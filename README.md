# Email Spam Analysis and Classification (NLP)

End-to-end NLP project to classify spam emails and extract insights from their content.

## Project Overview

This project focuses on building a pipeline to:
- classify emails as spam or non-spam
- analyze the main topics in spam emails
- measure semantic similarity between topics
- extract organizations from non-spam emails using NER

## Objectives

- Train a spam classifier
- Perform topic modeling on spam emails
- Compute semantic distances between topics
- Extract organizations from legitimate emails

## Dataset

Dataset provided as part of a training program:
https://github.com/ProfAI/natural-language-processing/tree/main/datasets/Verifica%20Finale%20-%20Spam%20Detection

**Note:** dataset not included in this repository.

## Workflow

- text preprocessing and cleaning
- feature extraction (TF-IDF / embeddings)
- spam classification model
- topic modeling (e.g. LDA)
- semantic similarity between topics
- Named Entity Recognition (NER) for organization extraction

## Technologies

- Python
- scikit-learn
- NLP techniques
- topic modeling
- NER (spaCy or similar)

## Key Results

- spam classification model trained and evaluated
- identification of main spam topics
- analysis of semantic relationships between topics
- extraction of organizations from non-spam emails
